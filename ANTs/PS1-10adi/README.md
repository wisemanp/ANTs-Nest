## Notes

The local file `PS1-10adi_lc_rest_UBVRIJH` contains absolute magnitudes in rest-frame time (days from peak), not apparent magnitudes with real MJDs.  It cannot be directly ingested into ANTs-Nest format without a rest-frame → observed-frame conversion (z = 0.203, MJD_peak ≈ 55470).  README written; no lightcurves/ file produced.
